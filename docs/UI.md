# MediaScraper UI 设计风格指南

> 关键词：**工具感 / 信息密集 / 安全可控 / 高效率**

***

## 1. 整体设计原则（Design Principles）

### 1.1 核心原则

| 原则   | 说明                 |
| ---- | ------------------ |
| 信息优先 | 元数据 > 装饰，表格为主      |
| 可控安全 | 所有“移动/覆盖”都有预览与二次确认 |
| 批量友好 | 默认支持多选 + 批量操作      |
| 状态可见 | 任何任务都能看到当前状态       |
| 一致克制 | 少动画、少颜色、少拟物        |

***

## 2. 视觉基调（Visual Tone）

### 2.1 风格定位

`Notion × Plex × 开发者工具`

* 偏 **暗色模式友好**

* 扁平化、低饱和

* 强对比用于“风险操作”和“状态提示”

***

## 3. 颜色系统（Color System）

### 3.1 主色调

| 用途         | 颜色      | 说明         |
| ---------- | ------- | ---------- |
| Primary    | #3B82F6 | 主按钮 / 高亮操作 |
| Secondary  | #64748B | 次要操作       |
| Background | #0F172A | 主背景（暗色）    |
| Surface    | #020617 | 卡片 / 表格背景  |
| Border     | #1E293B | 边框 / 分割线   |

> ⚠️ **默认暗色模式**，亮色模式可选（P2）

***

### 3.2 状态颜色（非常重要）

| 状态       | 颜色      | 场景   |
| -------- | ------- | ---- |
| 成功 / 已刮削 | #22C55E | 入库成功 |
| 进行中      | #FACC15 | 刮削中  |
| 警告 / 待补刮 | #FB923C | 文件新增 |
| 错误 / 失败  | #EF4444 | 匹配失败 |
| 未处理      | #64748B | 初始状态 |

⚠️ **状态 = 图标 + 颜色 + 文本（三件套）**

***

## 4. 字体与排版（Typography）

### 4.1 字体

| 用途      | 字体                           |
| ------- | ---------------------------- |
| UI 主体   | Inter / SF Pro / PingFang SC |
| 数字 / 路径 | JetBrains Mono / Fira Code   |

### 4.2 字号规范

| 层级      | 字号      |
| ------- | ------- |
| 页面标题    | 18–20px |
| 表头      | 13px    |
| 表格正文    | 13px    |
| 次要说明    | 12px    |
| 标签 / 状态 | 11px    |

***

## 5. 布局系统（Layout）

### 5.1 页面结构（通用）

`┌──────────────────────────────────┐`\
`│ 顶部导航（固定） │`\
`├──────────────────────────────────┤`\
`│ 工具栏（筛选 / 批量操作） │`\
`├──────────────────────────────────┤`\
`│ 主内容区（表格 / 列表） │`\
`├──────────────────────────────────┤`\
`│ 批量操作栏 / 状态栏（吸底） │`\
`└──────────────────────────────────┘`

### 5.2 间距系统

* 基础单位：**8px**

* 常用组合：8 / 16 / 24 / 32

***

## 6. 表格设计规范（核心）

### 6.1 表格视觉

`- 行高：40–44px`\
`- 无竖线，仅横线`\
`- hover 行高亮`\
`- 选中行有左侧高亮条`

### 6.2 表格行为

| 行为    | 规范           |
| ----- | ------------ |
| 单击    | 选中           |
| 双击    | 打开详情抽屉       |
| Hover | 显示快捷操作       |
| 多选    | Shift / Ctrl |

### 6.3 状态 & 完整性展示

`海报: 🖼️ / ❌`\
`NFO: ✅ / ⚠ / ❌`

> ❌ = 缺失\
> \
> ⚠ = 不完整 / 待校验

***

## 7. 按钮系统（Buttons）

### 7.1 按钮等级

| 类型        | 用途      |
| --------- | ------- |
| Primary   | 刮削 / 入库 |
| Secondary | 刷新 / 搜索 |
| Tertiary  | 文本按钮    |
| Danger    | 删除 / 覆盖 |

### 7.2 危险操作规则

`❗ 移动 / 覆盖 / 删除`\
`= 二次确认 + 明确影响范围 + 预览`

***

## 8. 弹窗 & 抽屉（Modal & Drawer）

### 8.1 使用场景

| 组件     | 使用场景        |
| ------ | ----------- |
| Modal  | 决策（匹配 / 确认） |
| Drawer | 信息（详情 / 日志） |

### 8.2 AI 消歧弹窗规范

`- 推荐项高亮`\
`- 展示置信度 & 理由`\
`- 允许一键改选`

***

## 9. 任务 & 进度反馈

### 9.1 全局任务中心

`- 所有任务可见`\
`- 可暂停 / 重试 / 取消`\
`- 失败任务必须有原因`

### 9.2 进度条规范

`[█████-----] 50%`

* 不使用无限 loading

* 超过 10s 显示 ETA

***

## 10. 图标系统（Icons）

### 10.1 图标风格

* 线性 / 扁平

* 推荐：**Lucide / Heroicons**

### 10.2 关键图标映射

| 语义 | 图标 |
| -- | -- |
| 刮削 | 🧲 |
| 入库 | 📥 |
| 剧集 | 📺 |
| 电影 | 🎬 |
| AI | ✨  |
| 警告 | ⚠️ |

***

## 11. 文案风格（Microcopy）

### 11.1 原则

* 明确、克制、可逆

* 避免“成功啦～”这种情绪化文案

### 11.2 示例

| 场景   | 文案                  |
| ---- | ------------------- |
| 刮削失败 | “未找到可靠匹配，请手动选择”     |
| 覆盖确认 | “将覆盖现有 NFO 文件（1 个）” |
| 补刮提示 | “检测到 2 个新剧集文件，尚未刮削” |

***

## 12. 可访问性（A11y）

* 对比度 ≥ WCAG AA

* 不仅靠颜色表达状态

* 表格可键盘操作

***

## 13. 禁止事项（Anti-pattern）

`✖ 自动移动文件无提示`\
`✖ 隐藏后台任务`\
`✖ 表格中滥用图标`\
`✖ 过度动画`\
`✖ 模糊状态（“处理中…”不说明）`

***

## 14. 可选扩展（P2）

* 亮色模式

* 多语言 i18n

* 紧凑 / 舒适 表格密度切换
